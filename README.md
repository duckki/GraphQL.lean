# graphql-lean

Lean models for GraphQL and, later, GraphQL federation.

Canonical GraphQL specification reference: [GraphQL September 2025 Edition](https://spec.graphql.org/September2025/).

For the project structure, dependency diagram, and module overview, see [docs/overview.md](docs/overview.md).

## Build

Build all Lean targets:

```sh
lake build
```

The main top-level libraries are:

- `GraphQL`
- `Federation`
